Programmieren 2 Projekt: Kniffel Spiel

Kompillieren und ausführen:
Im src Ordner befinden sich auch Eclipse Projekdateien, damit es einfach in Eclipse importiert werden kann. 
Zu beachten ist das Java 8 Features (Lambdas) genutzt wurden und JavaFX 8 für die GUI, daher ist Java 8 auf dem System erforderlich und in Eclipse muss e(fx)clispe installiert sein.
Dann kann von application/Main gestartet werden.
Fertig kompillierte .jar unter: https://github.com/Tech-13/HSAA-SS17-P2/releases

Spielen:
Nach dem eingeben der Spieler und klick auf den "Spiel starten" Button öffnet das Haupfenster. Dort kann mit dem Button unten links gewürfelt werden. Die Würfel können durch direktes anklicken markiert (vom neu würfel ausgenommen) werden. Mit einem Klick auf die entsprechende Zeile in der Tabelle werden die möglichen Punkte angezeigt, mit einem Doppelklick wird der Wert entgültig eingetragen. Danach ist automatisch der nächste Spieler an der Reihe.
